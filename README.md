## Backend

### `npm install`
### `npx sst start`

Deploy the infrastructure and starts the local Lambda development environment.

## Frontend

### `cd fronend/`
### `npm install`
### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser..
